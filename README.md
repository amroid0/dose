# DoseApp
app consist of 3 screens:
1. Login screen
2. List screen (load thelist of drugs returned from this api => https://run.mocky.io/v3/0b63b4fc-3f0e-49f6-adbc-3ddfd8cd3853)
3. Detail screen for shoing the details of drug

**Features:**
1. MVVM
2. Jetpack Compose
3. Hilt
4. Navigation 
5. Room
6. Retrofit
7. Unit Testing
